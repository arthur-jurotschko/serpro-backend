# Projeto de Integração com API SERPRO

## 📌 Descrição
Este projeto realiza consultas de CPF e CNH diretamente na API SERPRO, garantindo validação segura e eficiente.

## ⚙️ Tecnologias Utilizadas
- Node.js
- PostgreSQL
- Axios
- Express.js
- API SERPRO (OAuth2)

## 🏗 Arquitetura do Sistema
1. Autenticação via **OAuth2** na API SERPRO.
2. Consulta de **CPF/CNH** e retorno dos dados.
3. Armazenamento das requisições no **PostgreSQL**.
4. API REST pronta para futura integração com um **Front-end React.js**.
